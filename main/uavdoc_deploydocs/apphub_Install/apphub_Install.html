3. AppHub安装

AppHub安装

AppHub是一个提供数据展示及控制管理功能的JEE应用，应用以war包的形式部署在JEE服务器中（如tomcat）。

安装与配置

修改war中web.xml配置 > < param-name >uav.server.port< /param-name >
< param-value>AppHub启动端口< /param-value >

< param-name >uav.app.gui.redis.store.addr< /param-name >
< param-value>gui缓存(redis)地址< /param-value >

< param-name >uav.app.manage.redis.store.addr< /param-name >
< param-value>app管理信息缓存(redis)地址< /param-value >

< param-name >uav.app.ldap.redis.store.addr< /param-name >
< param-value>权限管理信息缓存(redis)地址< /param-value >

< param-name >uav.app.godeye.redis.store.addr< /param-name >
< param-value>监控数据信息缓存(redis)地址< /param-value >

< param-name >@url:uav.app.godeye.hbquery.http.addr< /param-name >
< param-value>AppHub管理服务的心跳数据查询服务地址< /param-value >

< param-name >@url:uav.app.godeye.healthmanager.http.addr< /param-name >
< param-value>AppHub管理服务的实时数据查询服务地址< /param-value >

< param-name >@url:uav.app.godeye.notify.view.http.addr< /param-name >
< param-value>画像与预警服务的预警中心服务地址< /param-value >

< param-name >@url:uav.app.godeye.notify.strategy.http.addr< /param-name >
< param-value>AppHub管理服务的实时预警服务地址< /param-value >

< param-name >@url:uav.app.manage.apphubmanager.http.addr< /param-name >
< param-value>AppHub管理服务的apphub服务地址< /param-value >

< param-name >@url:uav.app.upgrade.server.http.addr< /param-name >
< param-value>升级中心的服务地址< /param-value >

< param-name >@url:uav.app.apm.ivc.http.addr< /param-name >
< param-value>应用性能管理服务的调用链服务地址< /param-value >

< param-name >@url:uav.app.apm.jta.http.addr< /param-name >
< param-value>应用性能管理服务的线程分析服务地址< /param-value >

< param-name >@url:uav.app.hm.newlog.http.addr< /param-name >
< param-value>日志服务的日志服务地址< /param-value >

< param-name >uav.apphub.sso.ldap.connection.info< /param-name >
< param-value>LDAP配置信息< /param-value >

< param-name >uav.apphub.sso.grafana.connection.info< /param-name >
< param-value>Grafana配置信息< /param-value >

将war包拷贝到JEE服务器应用目录，如tomcat的webapps目录。

启动与登录

启动JEE服务器（例如Tomcat下 sh bin/start.sh），访问http://ip:port/apphub ，使用用户名admin密码admin登录AppHub。
