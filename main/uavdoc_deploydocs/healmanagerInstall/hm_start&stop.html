2.2 启动与停止

启动与停止

启动

启动前，需确认第三方环境均已启动。
AllInOne部署模式启动命令如下

cd uavhm sh bin/start.sh

分布式部署时，先启动心跳服务，然后启动其他服务。每个服务有对应的启动脚本，形如start_服务配置文件名.sh

sh bin/start_服务配置文件名.sh

通过jps命令可查看HM是否启动。

启动后查看服务日志

tail -f ../uavhm.metadata/logs/P.服务配置文件名.0.0.log

若日志无报错（出现E类型日志）则启动成功。

停止

服务启动时，会在用户crontab中加入值守脚本。若直接kill进程停止服务，服务会自动重启。如需停止服务,执行以下命令：

sh bin/stop_hm_服务名.sh (AllInOne部署则是sh bin/stop.sh)

若执行后进程仍未停止则kill进程即可。
